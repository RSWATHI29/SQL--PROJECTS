// Project Description
# 📊 Employee Performance Analysis Using SQL  

## 🔍 Project Overview  
This project analyzes employee performance and salary trends using SQL.  
It helps HR teams make data-driven decisions.  

## 🛠️ Technologies Used  
- SQL Server  
- Joins, Aggregations, Window Functions  
- Ranking & Performance Analysis  

## 📂 Files in This Repository  
- **schema.sql** → Table structure  
- **queries.sql** → SQL queries for insights  
- **dataset.csv** → Sample employee data  

## 🚀 How to Run This Project  
1️⃣ Download `schema.sql` and `queries.sql`.  
2️⃣ Open **SQL Server Management Studio (SSMS)**.  
3️⃣ Run `schema.sql` to create tables.  
4️⃣ Run `queries.sql` to get insights.
